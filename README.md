# Research Applications Platform

A professional platform for publishing and sharing interactive research applications developed using:

- R / Shiny
- Python
- MATLAB

## Features

- Interactive statistical applications
- Econometric and forecasting tools
- Time series analysis
- Machine learning applications
- Epidemiological modeling
- Financial and economic analytics
- Modern responsive interface
- Multi-language expansion ready

---

## Available Research Areas

| Field | Description |
|---|---|
| ARDL & NARDL | Econometric modeling and cointegration |
| Time Series | ARIMA, SARIMA, GARCH forecasting |
| Machine Learning | Neural networks and predictive analytics |
| Bayesian Statistics | Bayesian inference and MCMC |
| SEIR Models | Epidemiological simulation models |
| Financial Statistics | Risk analysis and volatility modeling |
| Multivariate Analysis | PCA, clustering, factor analysis |
